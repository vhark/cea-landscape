# CEA Landscape

[![Validate](https://github.com/vhark/cea-landscape/actions/workflows/validate.yml/badge.svg)](https://github.com/vhark/cea-landscape/actions/workflows/validate.yml)
[![Build & Deploy](https://github.com/vhark/cea-landscape/actions/workflows/build.yml/badge.svg)](https://github.com/vhark/cea-landscape/actions/workflows/build.yml)

An interactive, community-maintained map of the **Controlled Environment Agriculture (CEA)** technology ecosystem — covering sensors, control systems, software, and services for indoor farms, greenhouses, vertical farms, and cannabis cultivation facilities.

🌿 **[View the Landscape →](https://vhark.github.io/cea-landscape)**

---

## About

This landscape is modeled after the [CNCF Landscape](https://landscape.cncf.io) and built with [landscape2](https://github.com/cncf/landscape2). It is:

- **Open** — anyone can submit additions or corrections via PR or Issue
- **Community-maintained** — no vendor controls the curation
- **Technology-focused** — covers the full stack from sensors to software to services

---

## Categories

| Category | Description |
|---|---|
| 🌡️ Sensors & Monitoring | Climate, CO2, substrate, and water quality sensors |
| 🎛️ Control Systems | Environmental control, lighting, irrigation, and fertigation automation |
| 💻 Software & Analytics | Grow management, compliance, and data platforms |
| 🔌 Connectivity & Integration | Time-series databases, IoT middleware, and cloud platforms |
| 🏗️ Infrastructure & Facilities | Vertical farms, greenhouse construction, and facility services |
| 🛒 Marketplaces & Services | Equipment distributors, consulting, and integration services |

## Contributing

**Adding your tool takes ~5 minutes:**

1. Fork this repo
2. Add your entry to `data.yml`
3. Add your logo SVG to `logos/`
4. Open a Pull Request

Full instructions: [CONTRIBUTING.md](CONTRIBUTING.md)

Not comfortable with Git? [Open an issue](https://github.com/vhark/cea-landscape/issues/new/choose) instead.
